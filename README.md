# homebrew-vchewing

## Description

homebrew-vchewing is a Homebrew Cask to install [vChewing-macOS](https://github.com/vChewing/vChewing-macOS). 

## Usage

```shell
brew tap windwords/vchewing
brew install --cask vchewing
```
